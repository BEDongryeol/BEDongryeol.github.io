자바 배열 구현

Array 구현

  • jdk 클래스 : ArrayList, Vector

    구현 함수

  • 배열은 크기를 미리 정해주어야 한다.
    • ARRAY_SIZE = 배열의 크기
    • count = 배열 내 요소 개수
    • addElement = 요소 추가 함수
    • insertElement = 요소 삽입 함수
    • removeElement = 요소 제거 함수
    • getSize = 배열의 크기
    • getElement = 검색 함수
    • printAll = 모든 요소 출력
    • removeAll = 모든 요소 삭제
