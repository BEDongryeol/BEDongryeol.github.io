탐색 (Search)

종류

순차 탐색 (Sequential Search)

  • 특정한 원소를 찾기위해 순차적으로 탐색을 수행한다.

이진 탐색 (Binary Search)

  • 배열의 내부 데이터가 이미 정렬되어 있는 상황에서 사용한다.
  • 탐색 범위를 절반씩 좁혀가며 탐색을 수행한다.

    구성

  • start (가장 앞에 위치한 원소)
  • mid (중간에 위치한 원소)
  • end (가장 뒤에 위치한 원소)
